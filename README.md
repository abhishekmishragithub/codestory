## Getting Started

### Basic usage
```bash
codestory generate
```

### Specifying a model
```bash
codestory generate --model openai/gpt-3.5-turbo
```

### Including a detailed description
```bash
codestory generate --include-description
```

#### Using emojis
```bash
codestory generate --emoji
```
