## Getting Started

### Basic usage
```bash
codestory generate
```

### Specifying a model
```bash
codestory generate --model gemini/gemini-1.5-flash
```

### Including a detailed description
```bash
codestory generate --include-description
```

#### Using emojis
```bash
codestory generate --emoji
```
